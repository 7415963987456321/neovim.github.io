Roadmap

The roadmap and milestones define the project direction and priorities.

Next

0.3

  • Lua remote plugin host
  • Externalize more UI components
  • Basic support for externalized code navigation (LSP)

Future/Long-term

  • Embed Neovim everywhere
  • Externalize window layout ("Smart UI")
  • Syntax API: allow clients to query and define syntax groups, text attributes

Meta

There are release milestones and meta milestones ("buckets").

  • Version-number milestones (0.1, 0.2, 0.3, …) assign issues to production releases.
    • The nearest upcoming version is assigned a target date.
  • The todo bucket holds triaged tickets with "medium" (not urgent) priority.
  • The unplanned bucket holds low-priority items lacking a feasible path to completion.
