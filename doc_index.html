Documentation

User

To install Neovim use your system package manager or download a pre-built archive.

The Neovim user manual is available online or with the :help command.

  • Check the Neovim FAQ for solutions to common issues.
  • If you've used Vim before, see :help nvim-from-vim.
  • For a list of Neovim features see :help nvim-features.
  • Breaking changes are documented here.
  • See :help quickref and the Vim FAQ for detailed reference.
