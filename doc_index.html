Documentation

The following resources are available for users and developers.

User

Install

See Installing-Neovim on the wiki.

Help

The user manual is available online and in Neovim with the :help command.

  • For an overview of Neovim-specific features see :help nvim-features
  • To use python plugins (such as YouCompleteMe) see :help provider-python
  • :help quickref is a dense reference for regular users
  • Vim FAQ is an extensive collection of "how to" answers.
  • If you use Vim already see :help vim-differences

Troubleshoot

  • The Neovim wiki is the primary knowledge base for troubleshooting and tips.
  • Check the FAQ for solutions to common issues.
  • Breaking changes are documented at Following HEAD.

If you're stuck, reach out to the Neovim community for help.

Developer

Code

  • :help dev-help and the wiki are the primary sources of developer information.
  • Source code documentation generated by Doxygen

Static Analysis

  • PVS-studio analysis runs daily.
  • Clang-scan analysis runs on every commit.
  • Coverity static analysis runs daily. To get access, see the contribution guidelines.
  • Translation report page
